# Power BI Theme Creator

A powerful, modern web application for creating, customizing, and exporting Power BI themes. Built with React, TypeScript, and Tailwind CSS.

![Power BI Theme Creator Preview](./public/preview.png)

## üöÄ Features

*   **Color Harmony Generator**: Generate palettes using Monochromatic, Analogous, Complementary, and other color theory rules.
*   **Trending Palettes**: Browse and apply top trending color palettes fetched live from the community.
*   **Live Preview**: Visualize your theme on a mock Power BI dashboard with KPI cards, charts, and matrices.
*   **Dark/Light Mode**: Toggle between dark and light report backgrounds to see how your colors adapt.
*   **Advanced Customization**:
    *   **Font Family**: Choose from standard Power BI fonts.
    *   **Border Radius**: Adjust the roundness of visuals.
    *   **JSON Export**: Download a ready-to-use `.json` theme file compatible with Power BI Desktop.
*   **Accessibility**: Automatic contrast checking to ensure your reports are readable.
